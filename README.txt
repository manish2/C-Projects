What this application does:
---------------------------
Reads from custom data file and outputs 
students based on input 

How to test: 
------------
showid will show the id of the student 

show will show full details of the student 

- operator will specify descending order 

you can also specify name followed by student first and last name  (note * will match all names found) 

e.g. 

show name homer *
will show all students with first name homer regardless of last name
